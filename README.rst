README
======

Customizations
--------------

* ``mapleader`` is set to ``,``
* ``NERDTree`` can be toggled on and off with ``<leader>n``
* Consecutive statements can be aligned around the ``=`` operator with ``<leader>a=``
* Consecutive statements can be aligned around the ``=>`` operator with ``<leader>a=>``

Pathogen Bundles
----------------

This vim distro ships with the following bundles, which are managed by Pathogen:

* NERDCommenter
* NERDTree
* Supertab
* Syntastic
* Tabular
* Fugitive
